# JDBC Learning 🚀

This repository contains my hands-on practice and learning journey with JDBC (Java Database Connectivity).

## 📌 About

JDBC is a Java API that enables Java applications to interact with relational databases. This repository includes examples, programs, and mini-projects that I built while learning database connectivity and CRUD operations using Java and MySQL.

## 🛠️ Technologies Used

* Java
* JDBC
* MySQL
* Eclipse IDE
* Maven (where applicable)

## 📚 Topics Covered

* JDBC Architecture
* Database Connectivity
* DriverManager
* Connection Interface
* Statement
* PreparedStatement
* ResultSet
* CRUD Operations
* Exception Handling

## 🎯 Learning Objectives

* Understand how Java applications communicate with databases.
* Perform Create, Read, Update, and Delete (CRUD) operations.
* Write secure SQL queries using PreparedStatement.
* Manage database connections efficiently.
* Build a strong foundation for learning Hibernate and Spring Boot.

## 🚀 Progress

This repository is continuously updated as I learn new JDBC concepts and implement them through practical examples.
